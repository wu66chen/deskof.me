## 使用说明

1. 用上述提示词在 Gemini-3-Pro-Image-Preview 或 GPT-Image-2 中生成图片
2. 将生成的图片放入 `public/assets/` 对应目录
3. 在网站设置面板中更新图标 URL 或壁纸 URL
4. 图标推荐格式：PNG（支持透明背景）
5. 壁纸推荐格式：JPG（较小文件大小）
